Frequently Asked Questions (FAQ)

How do I use the map page?
Go to the map page from the main menu. Use the search bar to find a location, or click markers to see route info.
Can I search for specific routes?
Yes! Use the search input in the search or map page to search by location name or destination.
What should I do if the map doesn't load?
Make sure you are connected to the internet. If the issue continues, try refreshing the page or using a different browser.
Is this service free?
Yes, the route and map services provided on this platform are completely free to use.

Useful Tips

  • Bookmark the map page for quick access to routes.
  • Use Chrome or Firefox for the best performance.
  • Allow location access to get real-time directions (if supported).
  • Click any route name in the search results to highlight it on the map.
